# 盲点与风险清单

| 风险 | 当前状态 | 处理原则 |
| --- | --- | --- |
| 数据版权与公开范围 | `books/` 原文仅内部使用 | 投稿和开源不包含完整原文或大段受限内容 |
| 切块质量一致性 | 已有规则，但缺少系统抽样复核 | 建立抽样审核表，记录问题和修订原因 |
| 评测集代表性 | 目前只有 smoke 样本 | 正式实验前必须扩展并完成专家审核 |
| 答案是否被证据支持 | 自动程序不能可靠判断 | 需要人工/专家审核或明确标注为待核查 |
| 基线选择 | 目前只有原型和 lexical smoke 检索 | 后续加入 Qdrant/bge-m3 与必要消融 |
| 路径与环境可复现 | 旧脚本存在硬编码服务器路径 | 新 CLI 使用 `configs/project_config.json` |
